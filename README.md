# A CRUD based Employee-Management-System using ReactJS, NodeJS, ExpressJS and MySQL

- <h2>TechStack</h2>
  Make sure you have the following installed: <br>

  - Node.js (v14.0.0 or higher)

  - React (v16.8.0 or higher)

  - MongoDB<br><br>

  Clone the repository:

   ```bash
   git clone https://github.com/your-username/simple-crud.git

   To run frontend-
   1.cd client
   2.npm start

   To run the backend-
   1.cd server
   2.nodemon index.js
 - <h2>About</h2>

    This is a simple CRUD app using ReactJS, NodeJS, ExpressJS and MongoDB. <br>
